## 🛠️ 错误处理

### 网络错误
- 连接超时：30秒
- 自动重试机制
- 友好的错误提示

### API错误
- Token过期自动刷新
- 无效单号提示
- 服务不可用提示

### 用户输入错误
- 格式验证
- 数量限制（最多10个）
- 重复单号检测

## 📊 状态指示器

界面右上角显示API连接状态：
- 🟢 **API已连接**：成功连接到顺丰API
- 🟡 **API连接中**：正在连接或查询中
- 🔴 **API连接失败**：连接失败或查询出错
- ⚪ **API就绪**：初始状态，等待查询
